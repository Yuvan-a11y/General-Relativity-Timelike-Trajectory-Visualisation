# General-Relativity-Timelike-Trajectory-Visualisation

This repository contains Python scripts used in my essay:

**"A Computational Investigation of Timelike Curves and Embedded Geometry in General Relativity"**

## Contents
- Gödel spacetime null hypersurface visualization
- Timelike geodesic congruence plots
- Energy-density diagnostic plots
- Kerr frame-dragging visualization
- Embedded hypersurface construction

## Purpose
The code demonstrates a first-principles workflow:
metric → geometry → visualization

It is designed for **clarity and pedagogy**, not high-performance GR simulation.

## Requirements
- Python 3
- NumPy
- SciPy
- Matplotlib
- SymPy
